## Usage
**建议使用 >= Node v18**

![](https://img.cdn.sugarat.top/mdImg/MTY4NTU0MjM3NTI1Mw==685542375253)

### install dep
need install pnpm
```sh
# pnpm8要求 Node >= 16
npm i -g pnpm
```

```sh
pnpm install
```

### Dev
```sh
pnpm dev
```

### Build
```sh
pnpm build
```

### Preview Build
```sh
pnpm serve
```
